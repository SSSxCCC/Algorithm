### java自带：

哈希表HashMap、HashSet：快速映射去重，重写equals和hashcode

红黑树TreeMap、TreeSet：排序且去重，注意compareTo相等就去重，不看equals和hashcode

优先队列PriorityQueue：取最大/最小值

### 运行

1.下载源代码

2.使用IntelliJ IDEA打开项目

3.设置Java版本>=8

4.打开任意数据结构/算法的源代码，运行main函数
